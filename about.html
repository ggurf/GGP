Extra Details

This website was built using HTML and CSS, and developed in Visual Studio Code. I used GitHub to manage and back up my code, which also allowed me to keep track of changes and version history throughout the project. To make sure the code followed best practices, I used W3Schools’ HTML and CSS validators to check for any errors or inconsistencies. I also ran the site through the WAVE accessibility tool to ensure it’s usable for a wide range of users, including those with disabilities. These tools helped me build a more reliable, accessible, and professional-looking portfolio that meets modern web standards.
